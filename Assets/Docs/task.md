# HomeAssignment_1 (адаптация под 3D раннер)

## Общее описание
Создать минималистичный **3D раннер**, где игрок движется по бесконечной дорожке, уклоняется от препятствий и выживает как можно дольше.  
Визуал — примитивы Unity (Cube, Capsule, Plane).

## High-концепт (3D)
Игрок — примитив (Cube/Capsule), камера — сзади/сверху.

Игрок:
- Автоматически движется вперёд по оси Z  
- Двигается влево/вправо по оси X  
- Прыгает (не более одного прыжка в воздухе, возможен двойной)  
- Имеет здоровье  
- Получает урон при столкновениях  
- Умирает при 0 HP → игра перезапускается  

---

## Общие требования
1. Все параметры делаются настраиваемыми через Инспектор  
2. Если тип объекта спавнится много раз — статические данные хранятся в `ScriptableObject`  
3. Не должно быть необработанных исключений  

---

## Основные требования

### 1. Игрок
Игрок должен:
- Постоянно двигаться вперёд  
- Перемещаться влево/вправо  
- Прыгать (ограничение: один прыжок в воздухе)  
- Иметь максимальное и текущее здоровье  
- Получать урон при столкновениях  
- Умирать при 0 HP  

Можно использовать старую систему ввода; новая даёт +1 балл.

### 2. Препятствия
Нужно реализовать:
- Минимум два типа препятствий (разная форма + разный урон)  
- Периодическую генерацию перед игроком  
- Случайный выбор типа  
- Размещение по линиям: например X ∈ {−lane, 0, +lane}  
- Удаление препятствий, которые игрок прошёл
