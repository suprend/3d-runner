# 3D Runner — HomeAssignment_1

**Студент:** Поспелов Михаил

## Описание
Минималистичный 3D раннер на примитивах Unity: игрок автоматически бежит вперёд по оси Z, меняет линию по X, прыгает/скользит, получает урон от препятствий. При 0 HP показывается Game Over и доступен перезапуск.

## Точки входа
- **Главная сцена:** `Assets/Scenes/SampleScene.unity`
- **Главный скрипт:** `Assets/Scripts/RunnerBootstrap.cs` (создаёт игрока/трассу/спавнер на старте)

## Управление (New Input System)
- Move: `A/D` или `←/→` (переключение линий)
- Jump: `Space`
- Slide / fast-fall: `↓` (или удержание `S`)
- Wall break: `E`
- Restart (после Game Over): `R` или кнопка `Restart`

## Реализация (кратко)
- Все параметры настраиваются в `Assets/Configs/RunnerConfig.asset` (ScriptableObject).
- Типы препятствий заданы через `Assets/Configs/ObstacleType_*.asset` (ScriptableObject), спавн — случайный по линиям.
- Сложность растёт со временем: увеличивается скорость и уменьшается интервал спавна.
- Очки считаются по пройденной дистанции (по Z).
- При смерти показывается Game Over экран, рестарт по `R` или кнопке `Restart`.
- Прогрузка/перестройка мира скрыта через Linear Fog и fade при рестарте.

Примечание: `InputActionAsset` создаётся в рантайме (без `.inputactions` файла).
