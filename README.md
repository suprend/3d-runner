# 3D Runner — HomeAssignment_2

**Студент:** Поспелов Михаил

## Описание
Минималистичный 3D раннер на примитивах Unity. Игрок автоматически движется вперёд по оси `Z`, переключается между линиями по `X`, прыгает, скользит, собирает бонусы и избегает препятствий. Во время игры отображаются здоровье, очки, активные бонусы и рекорд. Между запусками сохраняется лучший результат.

## Точки входа
- Главная сцена: `Assets/Scenes/SampleScene.unity`
- Главный bootstrap-скрипт: `Assets/Scripts/RunnerBootstrap.cs`
- Управление состояниями игры, меню и паузой: `Assets/Scripts/RunnerGameManager.cs`

## Управление
- `A/D` или `Left/Right`: смена линии
- `Space`: прыжок
- `Down` или `S`: скольжение / fast-fall
- `E`: wall break
- `Esc`: пауза / продолжение
- `R` или `Enter`: рестарт после `Game Over`

## Реализация
- Все игровые параметры настраиваются через `RunnerConfigSO` в `Assets/Configs/RunnerConfig.asset`.
- Типы препятствий и бонусов вынесены в `ScriptableObject`: `ObstacleTypeSO` и `PickupTypeSO`.
- Рантайм-сцена собирается кодом в `RunnerBootstrap`: игрок, трасса, спавнер, камера и UI.
- Очки считаются по пройденной дистанции; рекорд сохраняется через `PlayerPrefs`.
- Главное меню, HUD, пауза и `Game Over` создаются кодом в `RunnerUIController`.
- Визуальный отклик урона и бонусов дополнен `Animator`-контроллером для ауры игрока. Контроллер автоматически создаётся editor-скриптом `Assets/Editor/RunnerFeedbackAnimatorAssetGenerator.cs`.
